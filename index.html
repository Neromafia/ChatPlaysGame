Cách Sao Chép ID TikTok

ID TikTok (hay Username) là tên duy nhất bắt đầu bằng chữ "@".

  1. Mở ứng dụng TikTok và vào tab "Hồ sơ" (Profile).
  2. ID của bạn chính là dòng chữ nằm ngay dưới ảnh đại diện.
  3. Nhấn vào ID đó (ví dụ: @chatplaysgame) để tự động sao chép.

Bạn chỉ cần Dán (Paste) ID vừa sao chép vào ô nhập. Trang web sẽ tự động xử lý.
